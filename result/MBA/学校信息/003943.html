2021 IMD MBA 就读体验 [2021.11.30]

来 IMD 之前，我认为 MBA 会是自己人生的转折。现在来看，选择 IMD，是我人生最重要和成功的决定之一。

一、套用某个师兄帖子的两个问题，再来一次：

a. 会不会读 MBA， 会！
b. 会不会选 IMD, 一定会！
c. 不读 MBA 能不能到现在的工作：零可能！

提示：体验和视角因人而异，多空自取；每个人都是有个体差异的，能动用和挖掘到资源，感受，体会也大有不同。

二、学习体验，完美符合 IMD 的理念： REAL IMPACT, REAL LEARNING

小班，多样性，非常紧凑，bounding 强，30 多还是 40 个国家（忘了），有趣的灵魂很多，实力强劲的人很多。能来的，都有自己特别突出和优秀的地方。

中国同学：

  • 有写小说的
  • 有边上学边运营公司的
  • 有周游欧洲听歌剧的
  • 有 hiking 遍布瑞士山川河流的
  • 有踏雪山湖水宿营的
  • 有广交朋友体验不同生活风格的

四、IMD 有趣的体验：

  1. 自然环境体验完美
  2. 周游欧洲和瑞士
  3. 喝酒，泡吧，户外烧烤（群体性）
  4. 运动：跑步，湖里游泳，帆船，羽毛球（团体大项目），足球，hiking，瑜伽，滑雪，跳伞，健身等等
  5. 以前的 discovery trip 是周游三个大洲，访问不同的国家和企业，但疫情原因，今年安排是瑞士山里团建一周和柏林一周行。相比以前略有遗憾，但这两周的集体性体验，也是不错的经历。

七、参与的咨询项目：

IMD 资源：

  1. 一个校友（是自动驾驶 START UP 的 CEO，之前成功带领另外一家公司上市完成了原始积累）开学后联系，希望我帮他们 review 系统设计，并对接一些国内供应商资源。零零散散，跟进了 3-4 个月。
  2. IMD-startup 项目：帮助 EPFL 的一个 startup 做小型医疗器械的 GO-TO-MARKET STRATEGY，零零散散做了 3 个月。
  3. 暑期实习资源：帮助一家德国公司做1亿欧元规模的基金投资机会分析，做了 1.5 个月全职，也为我的简历增加了欧洲的工作经历。
  4. 下半年 IMD-international consultingproject：北欧公司的 sales force 项目，完整 2 个月。

IMD 外的资源：

  3. 四大中的一家英国所：行业技术发展方向和市场规模分析
  4. 日本产品开发公司：行业中某一个应用端的市场规模以及未来发展趋势
  5. 还有零星的项目，有VC也有行业公司，基本都跟市场机会，规模，未来发展趋势相关

九、就业

这个是大家最关心的问题，但每年就业市场不一样，candidate 的 skill set 还有 professional 的经验不一样。但从今年总体结果来看，IMD 的就业非常不错。整体班级情况非常好，大部分同学都 focus 在欧洲市场，主要出路还是咨询和 MBA 项目，或者利用过去经验去 corporate。

中国同学就业也很好，咨询、MBA 项目、startup 等等。今年亚太区域找工作反而是供大于求，所以竞争激烈，可能会更吃之前的工作经验是否能 match。

抓取三个中国同学毕业前收获的 offer：

A：2 个咨询，1 个 healthcare 的 LDP
B：1 个工业的 LDP，1 个 tech 的 LDP
C：1 个咨询，1 个工业 offer，1 个 healthcare 的 LDP
这些 Offer 都能实现待遇、地点、行业、职能的转变。

十、要不要读 MBA

这是个经久不衰的讨论。 但真的因人而异。每个人情况不一样：工作经验，经济条件，个人能力，性格，语言，家庭状况等等。

建议大家读之前思考几个问题：

  1. 自己是否还有潜力尚未挖掘，需要MBA提供平台实现转变；
  2. 经济条件是否能支撑自己，即使失败，也能承受的住损失；
  3. 如果毕业回国，是否有清晰的目标可以 match 自己的经验（非互联网的，想去互联网 MBA 项目有难度，但也不是绝对）；如果留欧，自己是否有强有力的比较竞争优势胜过欧洲人或者英语母语的人。

总结：这一年的付出得到了回报，但更大的转变是一年中，IMD 对我在 mindset 和心态上带来的影响。IMD 提供了强大的资源供你去挖掘，每个人的收获也不尽相同。当然，IMD 也有的缺点和不足，但取决于你自己如何取其精华，然后融会贯通。

我想，当你真正发生转变之后，才会助推未来的职业发展。

祝大家好运！
